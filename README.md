# AnimalClasification
Parcial de IA 2 con CNN para clasificación de animales (oso, rata, cuervo y elefante)

para activar el entorno copien y peguen en la terminal:
.venv\Scripts\activate

si no tienen instalado las librerías corran en la terminal, luego de activar el venv:
pip install -r requirements.txt 

para correr en local pongan en la terminal:
python api.py
